### 커밋 컨벤션

| 타입 | 설명 |
| --- | --- |
| **feat** | 새로운 기능 추가 (기능 당 최초 한번만 사용) |
| **fix** | 버그 수정 |
| **docs** | READEME와 같은 문서 수정 |
| **style** | 코드 스타일 변경 (세미콜론 추카, 포맷팅 등) |
| **refactor** | 기능 변경 **없**는 코드 개선 |
| **test** | 테스트 코드 추가/수정 |
| **chore** | 빌드 설정 변경, 패키지 업데이트 등 (Gradle 설정) |
| **rename** | 파일 혹은 폴더명 수정하거나 옮기는 경우 |
| **remove** | 파일을 삭제하는 작업만 수행하는 경우 |

```
태그: 작업 내용 요약 #이슈 번호
- 설명(생략 가능)

ex)
feat: 로그인 기능 구현 #13
- 어쩌고 저쩌고
```
